🖼️ WebP와 AVIF란?

WebP는 구글이 개발한 이미지 형식으로, 웹 이미지의 파일 크기를 줄이는 동시에 높은 품질을 유지하도록 설계되었습니다.

AVIF는 AV1 비디오 코덱 기반의 최신 이미지 포맷으로, JPG나 PNG보다 훨씬 작은 파일 크기로 고품질 이미지를 제공합니다.

일부 오래된 소프트웨어나 기기에서는 WebP나 AVIF 형식을 지원하지 않을 수 있어, JPG나 PNG와 같은 보편적인 형식으로 변환하면 호환성 문제를 해결할 수 있습니다.

📋 변환 형식 선택 가이드

JPG 선택 시 좋은 경우:

  • 사진이나 복잡한 그래디언트가 있는 이미지
  • 파일 크기를 최소화하고 싶을 때
  • 약간의 품질 손실이 허용되는 경우

PNG 선택 시 좋은 경우:

  • 투명도(알파 채널)가 필요한 이미지
  • 텍스트나 선이 많은 이미지
  • 무손실 품질이 필요한 경우

WebP 선택 시 좋은 경우:

  • 웹에서 최적의 파일 크기와 품질이 필요할 때
  • 현대적인 브라우저 환경에서 사용하는 경우
  • 최신 이미지 포맷을 원하는 경우

AVIF 선택 시 좋은 경우:

  • 최신 브라우저에서 극도로 작은 파일 크기가 필요할 때
  • 최고 품질의 압축을 원하는 경우
  • 최신 형식 지원이 가능한 환경
